Conheça a nossa equipe

Somos uma equipe dedicada ao desenvolvimento de um projeto inovador de chatbot, com o objetivo de transformar a interação entre usuários e tecnologia.

Como foi feito?

Desenvolvimento do projeto

Configuração Inicial do Projeto

Reunimos o grupo e o planajamento da configuração de ferramentas que iriamos utilizar;

Todos os integrantes participaram dessa etapa

Desenvolvimento do Backend

Desenvolvemos o backend com Node.js e Express, criando as rotas e funcionalidades necessárias. Integramos a API do Gemini para processar as respostas do chatbot.

Principais responsáveis: Tiago Veloso Macedo e Guilherme Saccomani

Design do Frontend

Criamos a interface do usuário com HTML e CSS, focando em um design intuitivo. Integramos o frontend ao backend para uma comunicação eficaz.

Principais responsáveis: Zelio Horta Dias, Marco Antônio e Vitor Calandrim

Hospedagem Local e Testes

Hospedamos a aplicação em um servidor local para testes. Realizamos verificações para garantir o bom funcionamento do chatbot antes do deploy.
